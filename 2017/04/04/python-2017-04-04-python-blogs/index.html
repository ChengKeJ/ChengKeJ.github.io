今天一时兴起，想用python爬爬自己的博客，通过数据聚合，制作高逼格的云图(对词汇出现频率视觉上的展示)，看看最近我到底写了啥文章。

二、技术栈

  • 爬虫技术：bs64、requsts、jieba
  • 分析工具：wordArt

三、爬虫构架设计

整个爬虫架构非常简单：

  • 爬取我的博客：http://blog.csdn.net/forezp

  • 将数据用“结巴”库，分词。
  • 将得到的数据在在artword上制作云图。
  • 将制作出来的云图展示给用户。
